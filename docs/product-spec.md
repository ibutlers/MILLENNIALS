# Especificación de producto — Realstate

Realstate será una webapp inmobiliaria para presentar, consultar y gestionar oportunidades de real estate. Esta tarea prepara repositorio, reglas, scripts, pruebas y deploy; no implementa el producto completo.

## Usuarios
- Visitantes interesados en propiedades.
- Operadores que gestionan inventario.
- Administradores del sistema.

## Alcance de esta preparación
- Landing mínima verificable.
- API mínima con `/health`.
- Base técnica documentada y desplegable.

## Fuera de alcance
Autenticación real, panel administrativo, CRM, pagos, mapas, modelo definitivo y pantallas finales.
