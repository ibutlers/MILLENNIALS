# Realstate

Webapp Realstate.

## Stack
React + Vite + TypeScript, React Router, Tailwind CSS, TanStack Query, React Hook Form, Zod, Node.js + Fastify, PostgreSQL, Vitest, Testing Library, Playwright, Docker Compose + Caddy.

## Comandos
```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
pnpm test
pnpm test:e2e
pnpm build
./scripts/deploy.sh
./scripts/healthcheck.sh
./scripts/rollback.sh
```

Ver `AGENTS.md` y `docs/` antes de modificar el proyecto.
